# Curvify frontend

#### Description

Frontend for the curvify website

React typescript
MUI

### Warning

Developed and tested on macOS Sonoma

## Instructions

### pre-requisites

Install homebrew

### Step 1

Clone repository

```
git clone
```

### Step 2

Install npm

```
brew install npm
```

### Step 3

Install bundles using npm

```
npm install
```

### Step 4

Start frontend

```
npm start
```
