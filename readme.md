# Curvify frontend


#### Description:
Frontend for the curvify website  
uses MUI  
works with backend: github.com/travisim/Curvify-Backend/

## Warning 
Developed and tested on macOS Sonoma

## Instructions
### pre-requisites
Install homebrew  

### Step 1  
Install npm 

```
brew install npm
```

### Step 2  
Install bundles using npm

```
npm install
```

### Step 3
Start frontend

```
npm start
```
